## Features
- 非常好看的 HTML5 移动端音乐播放器，支持播放列表、播放模式（循环播放、随机播放等）、音量调节、进度控制，兼容现代浏览器和移动端，API 丰富、可扩展性强，无任何依赖。
- 技术栈：原生js + Ajax + CSS3 + Html5 + Flex
